# Job Finder Website (Symfony)

This is a web application built with Symfony that helps users find and apply for job opportunities بسهولة.

## 🚀 Features
- Browse job listings
- Search jobs by category or keywords
- View detailed job descriptions
- User-friendly interface
- Responsive design

## 🛠️ Tech Stack
- Symfony (PHP framework)
- HTML, CSS, JavaScript
- MySQL (database)

## 📌 Purpose
This project was developed to simplify the job search process by providing a centralized platform for job seekers.

## ▶️ How to Run
1. Clone the repository
2. Run `composer install`
3. Configure `.env` file (database connection)
4. Run `symfony server:start`
5. Open in browser: `http://localhost:8000`
